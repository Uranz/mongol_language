# Mongol Language Learning App

This is a web application I built to help beginners learn Mongolian language basics.

## Overview
- Flask backend (Python)
- SQLite database
- HTML / CSS frontend

## Features
- User login system
- Lessons with vocabulary and quizzes
- Database-driven architecture

## Tech Stack
- Python (Flask)
- SQLite
- HTML / CSS

## Status
This is an ongoing project designed to expand access to Mongolian language practice. Future work includes improving UI, adding more lessons, and speech features.
